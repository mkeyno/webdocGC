The hype around the Internet of Things concept has been pulsing heavy each year and is quickly materializing in real world form
simple home automation to the vast industrial applications. Keyno Guidance & Control solutions are base on IOT technologies
which connect devices together with simple WiFi node.
A WiFi module has been embedding in our driver and controller board which can selected as Access Point or Station. A
lightweight webhost installed in each module so user can easily connect to device and change desired module setting. Currently
our design can support all type TCP/UDP package, MQTT, and ModBus protocol. The main strength of our design come as
following
           •Each device has its own IOT portal which can configure device setting as well as wifi setting
           •User friendly web server for monitor device status and configure device and wifi setting 
           •Designed as shield to install on original board of each devices
            •Can connect to any BMS  standard package
           •Easily add other devices or sensor in to the network group
           •Data logging for energy monitoring and system debug analyses
           •User scope limit with ssl password protestation
          •Total solution for time-driven or event-driven scenario
